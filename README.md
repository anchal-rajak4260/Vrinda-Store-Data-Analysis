# Vrinda-Store-Data-Analysis
# 🛍️ Vrinda Store Annual Report - 2026

A data analytics project built entirely in **Microsoft Excel** using **Pivot Tables**, **Pivot Charts**, and **Slicers** to analyze Vrinda Store's 2026 sales performance across channels, categories, states, and customer demographics.

---

## 📊 Dashboard Overview

The interactive Excel dashboard provides a 360° view of store performance with the following visualizations:

| Chart | Type | Insight |
|-------|------|---------|
| Orders & Sales | Bar + Line (combo) | Monthly order count vs revenue trend |
| Sale: Men vs Women | Pie Chart | Women drive 78% of total sales |
| Age vs Gender | Grouped Bar | Adults are the top buying segment |
| Orders by Channel | Donut Chart | Amazon leads at 40%, Flipkart 26% |
| Sales: Top 5 States | Horizontal Bar | Madhya Pradesh tops at 0.18M |
| Order Status | Pie Chart | 92% of orders successfully delivered |

---

## 🔍 Key Insights

- 👩 **Women account for 78%** of all purchases vs 22% men
- 🏆 **Amazon is the #1 channel** (40%), followed by Flipkart (26%) and Myntra (22%)
- 📍 **Madhya Pradesh** is the highest revenue state at ₹0.18M
- 🧑 **Adult women** are the dominant customer segment (40.72%)
- ✅ **92% delivery success rate** with only 4% returns and 3% cancellations
- 📅 **Peak sales months** visible via the Orders & Sales combo chart

---

## 🛠️ Tools & Techniques Used

- **Microsoft Excel 2021 / Office 365**
- **Pivot Tables** — for aggregating sales by month, channel, state, category, and gender
- **Pivot Charts** — bar, pie, donut, and combo charts built directly from pivot data
- **Slicers** — interactive filters for Month, Category, and Channel
- **Data Cleaning** — handled missing values, standardized category names
- **Conditional Formatting** — for quick visual highlights

## 🧹 Data Cleaning Steps

1. Removed duplicate order entries
2. Standardized gender values (`M` → `Men`, `W` → `Women`)
3. Filled missing category fields
4. Converted date columns to proper `DATE` format
5. Created derived columns: `Month`, `Age Group`, `Order Status`

---

## 📌 Dashboard Filters (Slicers)

The dashboard includes three interactive slicers that dynamically update all charts:

- **Month** — Jan through Dec
- **Category** — Blouse, Bottom, Ethnic wear, Kurta, Saree, etc.
- **Channel** — Ajio, Amazon, Flipkart, Meesho, Myntra, Nalli, Others

---

## 💡 Business Recommendations

> Based on the dashboard analysis:

- Target **adult women aged 25–45** for marketing campaigns
- Focus ad spend on **Amazon and Flipkart** — together they account for 66% of orders
- Run promotions in **Madhya Pradesh, Maharashtra, and Karnataka** — top 3 revenue states
- Investigate the **4% return rate** to reduce losses, especially in high-volume categories
